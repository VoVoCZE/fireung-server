FIREUNG server v3 - oprava online kolizí

Co se mění:
- Online hráč už nemůže vizuálně lézt do beden/zdí.
- Server nově kontroluje celý hitbox postavičky, ne jen střed políčka.
- Public/private lobby systém zůstává stejný jako v2.

Nasazení na Render:
1) Nahraj tyto soubory do GitHub repozitáře pro fireung-server.
2) Na Renderu otevři službu fireung-server.
3) Dej Manual Deploy -> Deploy latest commit.
4) Počkej, až bude status Live.
5) Na fireung.eu dej Ctrl+F5.

Endora/index.html se kvůli této opravě měnit nemusí.
